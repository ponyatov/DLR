TUI: Text User Interface

As we are going to use DLR for RealTime control systems , graphics interfaces are not suitable for us. Even simple graphics like text output bluerequire a huge amount of memory to store window region images, blueand a lot of bus time to transfer (bitblit) this regions. Typical PLC or control panel can have tiny text-only LCD with down to short one line and few joystick-like buttons. No large touchscreen, qwerty keyboard or trackball devices.
