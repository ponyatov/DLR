There is no compiled/interpreted language

Programming language blueis formal specification of syntax and semantics, it is a documentation artifact, and this bluespecification can't be compiled1.2 !

For example, pure C is widely known as ``compiled language'', but Fabrice Bellard's http://bellard.org/tcc/Tiny C Compiler can run as interpreter (but uses dynamic compilation ).

Java looks like classical compiler suite with edit/compile/debug loop, but in fact blueJVM is interpreter by design, blueexpanded by JIT for programs speed up using dynamic compilation .
