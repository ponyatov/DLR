Virtual Machine Architecture

FVM12.2 has one byte-addressed memory, and two separate stacks:

[nosep]
data stack
for data
return stack
return addresses for call/ret commands


Sizes of this structures was defined by constants, but you can modify code and use expandable storage type like vector12.3



FORTH has special CELL constant, corresponds to machine word size in bytes.
